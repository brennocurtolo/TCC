# TCC
Implementação utilizada para o desenvolvimento do meu TCC.

## Configuração
Para configurar os Hiperparâmetros da rede, entre no diretório do modelo e altere diretamente o arquivo 'parameters.py'

## Execução
Para execução do código, navegue até o diretório do modelo e então execute o comando desejado:

```console
python (seguido do modelo que deseja ser testado)

python vit_CPU_DYNAMIC_LR_WARMUP.py
python vit_labelSmoothing.py
python vit_CPU_DYNAMIC_LR.py
python vit_CPU.py
python vit_CutMix.py
python vit_GPU.py
python vit_labelSmoothing.py
```

## Referências
Na pasta referências você irá encontrar o PDF do meu TCC e algumas documentações utilizadas. Demais referências estarão disponível diretamente nas referências do projeto.
